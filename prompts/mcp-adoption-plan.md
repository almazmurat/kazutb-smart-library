
Я использую AI для разработки сайтов и ещё не подключил MCP, я теряю половину возможностей. 3 MCP-сервера, которые реально меняют процесс разработки: автоматическое тестирование, генерация UI и работа с кодом.

Browser MCP тестирует сайт как человек — кликает, заполняет формы, ищет баги и делает скриншоты. 21st Devs Magic MCP генерирует готовые UX-компоненты по текстовому описанию. А Get MCP даёт ИИ доступ к коду и документации из GitHub в реальном времени.

В итоге ты не просто генерируешь код, а строишь полноценный AI-воркфлоу разработки.

Set up MCP support for this repository in a safe phased way.

Evaluate and, if appropriate, prepare integration for:
- GitHub MCP server
- Browser MCP
- a frontend UI generation MCP such as 21st Magic or equivalent
- repository/test/tooling MCPs

Important:
- Do not connect everything blindly.
- Start with the safest and most useful MCP configuration first.
- Document what was added, what was postponed, and why.
- Keep all changes minimal and reversible.
