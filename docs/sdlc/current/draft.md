# Full visual reset — Digital Library

## Problem statement
The current redesign is still too close to the old interface DNA:
- dark, heavy hero blocks
- gradient-led composition
- marketing/landing-page tone
- heavy CTA emphasis
- too much inherited structure from the previous site

That is **not** the target.

## Target outcome
Create a **true system-level visual reset** for Digital Library so the platform feels like a premium, calm, highly usable, operational university product — not a brochure site and not a generic SaaS template.

The visual language must feel:
- modern
- academic
- light
- editorial
- restrained
- trustworthy
- product-grade
- search-first and discovery-first

## Grounded design source
Use the verified Stitch project as the design source of truth:
- project: `Digital Library Platform`
- project id: `4601252383613536784`
- direction: `The Academic Curator` / `The Living Archive`

## Scope
The reset must cover the real platform surfaces:
1. Homepage
2. Catalog
3. Book detail
4. Resources page
5. Teacher workflow / shortlist workbench
6. Login
7. Account / member dashboard
8. Future librarian/admin direction

## Core requirements
- Do **not** preserve the old visual style by default.
- Do **not** create a startup SaaS landing page.
- Do **not** add fake metrics, testimonials, or decorative filler.
- Do **not** optimize for beauty at the cost of workflow clarity.
- Make the catalog feel like the core product surface.
- Make teacher flows task-oriented and professional.
- Make account/dashboard areas feel operational and role-aware.
- Clearly distinguish:
  - physical holdings
  - local digital materials
  - licensed external academic resources

## Page/system goals
- Homepage must be a functional product entry point with strong search.
- Catalog must become the main working interface with strong filters and result clarity.
- Book detail must feel bibliographic, structured, and actionable.
- Resources must be utility-first, not brochure-first.
- Teacher shortlist/workbench must support real course literature preparation.
- Admin direction must prepare for denser operational workspaces.

## Success condition
The new interface should make the current platform feel obsolete: more mature, more coherent, more useful, and more academically credible.
