# Session Protocol — KazUTB Library Platform

## How to Start Every Session

### Step 1 — Orient
Open [[START_HERE]] and read it.

### Step 2 — Check State
Open [[CURRENT_STATE]] and read the current status snapshot.

### Step 3 — Check Open Questions
Open [[OPEN_QUESTIONS]] — pick one to work on or add new ones.

### Step 4 — Check Tasks
Read [[CURRENT_STATE]] task board section.

---

## How to End Every Session

### Step 1 — Update Current State
Edit [[CURRENT_STATE]]:
- Update what changed today
- Update task statuses
- Add new tasks if found
- Remove completed tasks

### Step 2 — Log Decisions
Any decision made today goes into [[DECISIONS]] with date and reasoning.

### Step 3 — Clear Inbox
Process or delete everything in `03-inbox/`.
Never leave the inbox full.

### Step 4 — Update Task Log
Add a one-line entry to [[TASK_LOG]] summarizing the session.

---

## Emergency Context Recovery

If you need to reconstruct project context from scratch:
1. Read [[PROJECT_CONTEXT]] — full read
2. Read [[CURRENT_STATE]] — where we are now
3. Read [[DECISIONS]] — what was decided and why
4. Read [[OPEN_QUESTIONS]] — what is still uncertain

That is enough to resume any task.
