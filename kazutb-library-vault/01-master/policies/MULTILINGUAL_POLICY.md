# MULTILINGUAL_POLICY
> Derived from [[PROJECT_CONTEXT]] §28

## Required UI languages
- Russian — primary
- Kazakh — required secondary
- English — required tertiary

## Metadata rules
Titles, author names, abstracts, and keywords should support multilingual variants where available. UDC descriptions should reflect the current UI language.

## Search behavior
Search must work across Cyrillic and Latin input and respect the user's selected language without forced transliteration.

## Default language behavior
Default language is Russian. Guest language can be auto-detected from the browser and users should be able to save their preference.
