| Rule | Value |
|---|---|
| Maximum active reservations per user | 3 |
| Reservation lifespan after confirmation | 3 days |
| Auto-expire | Yes |
| Can a user cancel their own reservation? | Yes |
| Can staff override limits? | Yes |

## Loan rules
| Rule | Value |
|---|---|
| Standard loan period | 14 days |
| Reference-material period | 1 day / in-library only |
| Maximum active loans | 5 |
| Renewal | Once, if no active reservation exists |
| Overdue blocking | Yes, new reservations are blocked when overdue |

## Business interpretation
Reference and lending funds may have different configured behavior, but the system default is conservative and circulation-aware.
