# DIGITAL_MATERIALS_POLICY
> Derived from [[PROJECT_CONTEXT]] §19

## Non-negotiable rules
- no free download
- no direct file URLs
- text copying should be restricted as much as technically possible
- full reading requires authenticated access and the correct access flag

## Access table
| User type | Cover | Preview | Full read |
|---|---|---|---|
| Guest | ✅ | Limited | ❌ |
| Member | ✅ | ✅ | Only if `member_access` |
| Librarian | ✅ | ✅ | ✅ |
| Admin | ✅ | ✅ | ✅ |

## Current phase vs future phase
The current requirement is to build the protected storage model, metadata, upload flow, access flags, and basic viewer skeleton. Stronger watermarking and harder viewer controls are future enhancements.
