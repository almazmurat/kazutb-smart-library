## What the project is
KazUTB Library Platform is a new independent digital library platform for the university. It is not a decorative redesign and not a CRM shell; it is meant to replace the old MARC-SQL environment as the real operational foundation.

## Why a new system is needed
The prior environment did not publish the catalog well, did not support modern user workflows, and inherited poor migration quality. The new platform must support real discovery, access, operations, cleanup, and reporting.

## Project nature
This initiative is simultaneously:
- a product project
- a library operations project
- a digital transformation project
- a data reorganization and quality improvement program

## Non-goals
The platform is not a free file host, not a mockup, not a simple UI refresh, and not a system where CRM owns the library domain.

## Strategic target
The stated target is ambitious and clear: build the best university library platform in Kazakhstan while preserving real library meaning and production readiness.
