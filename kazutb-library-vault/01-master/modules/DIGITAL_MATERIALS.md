# DIGITAL_MATERIALS
> Derived from [[PROJECT_CONTEXT]] §19

## Module scope
This module manages cover images, protected digital attachments, access flags, and the controlled reading experience for library-owned digital materials.

## Current state
The library currently has a small digitized collection, with scanning capacity available for gradual expansion.

## Required capabilities
- upload cover images
- upload protected digital files
- assign access level per item
- show limited preview publicly and full reading only when allowed

## Governance
Access behavior is defined by [[DIGITAL_MATERIALS_POLICY]] and implemented over [[FILE_STORAGE_MODEL]].
