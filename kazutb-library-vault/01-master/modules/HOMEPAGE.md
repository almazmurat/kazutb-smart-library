# HOMEPAGE
> Derived from [[PROJECT_CONTEXT]] §17

## Required sections
The homepage should include a strong hero, search entry, featured collections, news, external resources, library services, location information, login CTA, trust metrics, and quick links.

## Visual strategy
The current phase requires a premium clean structure with strong typography and clear hierarchy. Motion-heavy enhancement such as video or 3D can come later, but base function must not depend on animation.

## Role in the platform
The homepage is the public face of the new library and should drive users toward catalog discovery and authenticated tools.
