# MEMBER_DASHBOARD
> Derived from [[PROJECT_CONTEXT]] §10 and §11

## One shared dashboard
Student, teacher, and employee all use the same member dashboard with identical access scope.

## Sections
| Section | Purpose |
|---|---|
| Profile | AD-sourced identity block |
| Active reservations | Current statuses and pickup state |
| Borrowing history | Past and current loans |
| Personal literature shortlist | Saved books and resources |
| Digital materials | Accessible controlled content |
| Notifications | In-app feed |
| Quick links | Catalog, resources, contact |

## Personal shortlist feature
The shortlist is a personal wishlist or draft reading list, not a formal syllabus-approval system.
