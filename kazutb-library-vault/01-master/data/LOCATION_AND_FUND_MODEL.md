# LOCATION_AND_FUND_MODEL
> Derived from [[PROJECT_CONTEXT]] §6.7

## Physical locations
The library operates across two physical university locations:
- Economic Campus Library
- Technology Campus Library

## Fund ownership model
The system must distinguish:
- college fund
- university-economic fund
- university-technology fund
- general library fund

## Public visibility vs staff visibility
Public users should see branch name, sigla, shelf location, and availability. Staff also see acquisition, accounting, and fund-reporting details.

## Precise location model
Each copy needs branch, room or cabinet, shelf reference, and sigla or storage code so it can be located operationally and shown accurately in the catalog.
