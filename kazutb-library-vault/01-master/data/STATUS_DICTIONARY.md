# STATUS_DICTIONARY
> Derived from [[PROJECT_CONTEXT]] §14

## Copy / item status
| Status | Meaning |
|---|---|
| `available` | On shelf and usable |
| `reserved` | Reserved for pickup |
| `issued` | Currently borrowed |
| `overdue` | Return deadline passed |
| `in_processing` | Being cataloged or labeled |
| `lost` | Reported lost |
| `under_repair` | In repair or rebinding |
| `digitization_pending` | Sent for scanning |
| `restricted` | Special or reference-only |
| `archived` | Removed from active fund |

## Reservation status
| Status | Meaning |
|---|---|
| `pending` | Waiting for librarian confirmation |
| `confirmed` | Confirmed by librarian |
| `ready_for_pickup` | Prepared for the user |
| `fulfilled` | User picked up the item |
| `cancelled` | Cancelled by user or staff |
| `expired` | Auto-expired after deadline |

## Scientific work status
`draft`, `under_review`, `approved`, `rejected`, `published`, `archived`

## News status
`draft`, `scheduled`, `published`, `archived`

## Contact message status
`open`, `in_review`, `resolved`, `archived`

## Digital material access levels
`none`, `cover_only`, `preview`, `member_access`, `staff_only`, `restricted`
