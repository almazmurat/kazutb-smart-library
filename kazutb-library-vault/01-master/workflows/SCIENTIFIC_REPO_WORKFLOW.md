# SCIENTIFIC_REPO_WORKFLOW
> Derived from [[PROJECT_CONTEXT]] §20.4

## Workflow
```
Author agrees with librarian
  → librarian uploads the work and fills metadata
  → status: draft
  → librarian submits for approval
  → status: under_review
  → library head or admin reviews
  → approved → admin publishes → status: published
  → or rejected → librarian revises and resubmits
```

## Key governance rules
- authors do not upload directly
- librarians moderate the intake
- admin or library leadership gives final publication approval
- guests see metadata only, while authenticated users can read approved content in a controlled viewer
